## 🎮 Destaque — Jogos em C#

Uma das partes mais interessantes deste repositório: jogos desenvolvidos do zero em C#, aplicando conceitos de lógica de programação, estruturas de controle e orientação a objetos na prática.

Desenvolver jogos é uma das formas mais eficazes de consolidar fundamentos de programação — cada jogo exige raciocínio lógico, controle de fluxo, manipulação de dados e tratamento de entradas do usuário.

---

## 📚 Conteúdos Abordados

| Tópico | Descrição |
|---|---|
| Variáveis e tipos de dados | Declaração, atribuição e conversão de tipos em C# |
| Estruturas condicionais | `if`, `else`, `switch` — tomada de decisões |
| Laços de repetição | `for`, `while`, `do-while` — iterações e controle de fluxo |
| Vetores e matrizes | Manipulação de arrays unidimensionais e bidimensionais |
| Funções e métodos | Modularização de código, parâmetros e retorno |
| Orientação a objetos | Classes, objetos, encapsulamento e herança |
| Desenvolvimento de jogos | Lógica de jogo, interação com o usuário via console |

## ▶️ Como executar

**Pré-requisitos:** .NET SDK instalado ([download aqui](https://dotnet.microsoft.com/download)) e Visual Studio ou VS Code.

```bash
# Clone o repositório
git clone https://github.com/alexsander020/Algoritmos-e-Logica-de-Programacao.git

# Acesse a pasta do projeto desejado
cd "Jogos em C# Feito em Aula"

# Execute o projeto no Visual Studio ou via terminal
dotnet run
```
